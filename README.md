# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. The website is designed to introduce myself, showcase my profile, and provide an easy way for visitors to get in touch.

## Features

* Responsive and clean design
* Home page with personal introduction
* Contact page for communication
* Simple and easy-to-navigate layout
* Built using only HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to practice front-end web development using HTML and CSS while building a personal portfolio website.
